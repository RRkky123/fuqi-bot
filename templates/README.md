# 圖片模板說明

此資料夾存放平安圖底圖模板。

## 命名規則

```
{condition}_{time_slot}.jpg
```

- condition: `sunny` / `cloudy` / `rainy`
- time_slot: `morning` / `afternoon` / `evening`

## 範例

- `sunny_morning.jpg`   — 晴天早安圖
- `rainy_afternoon.jpg` — 雨天午安圖
- `cloudy_evening.jpg`  — 陰天晚安圖
- `default.jpg`         — 萬用備援模板

## 規格

- 尺寸：1024 × 1024 px
- 格式：JPG（品質 85）
- 版權：建議使用 Unsplash / Pexels 的 CC0 免費圖庫，或自行設計
- 風格：暖色系、祥和感，適合長輩族群
